Desenvolvimento Front-End

O desenvolvedor Front-End é o profissional que cuida da parte do código que é interpretada diretamento pelo navegador.

No desenvolvimento por camada, HTML é o responsável pela marcação de conteúdo, o CSS cuida do estilo visual e o JavaScript é o responsável pelo comportamento.
